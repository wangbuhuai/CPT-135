Please write a Python program that calculates the volume of a cuboid.  Your program should have the following functionality in execution:

•Ask the user to enter the length of the cuboid, which should be an integer.

•Ask the user to enter the width of the cuboid, which should be an integer.

•Ask the user to enter the height of the cuboid, which should be an integer.

•Calculates the area of the cuboid.  Please make sure that you use the correct math formula.

•Show the calculated result (volume) in the console.

Please note that the user cannot see or understand your code.  Therefore, you should provide sufficient screen prompt to let the user know what to input.

Your program can assume that the user will always provide valid input values.  You do not need to worry about invalid input or error handling.
